Ben Overmyer

Full-Stack Developer and Game Designer

Hi, I'm Ben. I do web things and make games.

By day, I apply my skills as a developer to build web applications, monitor them, set up web infrastructure, and automate things.

In my free time I work on procedural content generation tools for tabletop role-playing games. You can see my latest works in this area at Iron Arachne.

I'm also one of the owners of tabletop role-playing game company Silver Gryphon Games.

You can find me on the Fediverse at @dungeonHack@fosstodon.org.
